Uitleg

Een favicon bundel bestaan vaak uit veel bestanden die allemaal apart van elkaar moeten worden ingeladen. Onderstaand uploadveld maakt dit makkelijk door een zip-bestand te accepteren met daarin alle verschillende favicon-bestanden. Het script zal dit zip-bestand uitpakken op de correcte locatie en de code genereren zodat deze netjes worden ingeladen

Het zip-bestand kan gegenereerd worden door een afbeelding te uploaden naar https://realfavicongenerator.net/

Data

Upload een zipbestand met daarin de favicon bestanden. Deze kan gegenereerd worden via https://realfavicongenerator.net/
