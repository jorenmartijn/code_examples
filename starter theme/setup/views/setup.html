Introductie

Dit is het overzicht van alle functionaliteiten die je kunt opzetten van de Nordique Project Setup. Veel van deze functionaliteiten verzorgen automatische generatie van WordPress onderdelen of het genereren van code snippets die automatisch aan je thema worden toegevoegd.
Elke plek in het thema waar gegenereerde code kan worden toegevoegd herken je aan het 'BEGIN [code]' en 'END [code]' stuk. Wil je dat er geen generereerde code meer kan worden toegevoegd, verwijder die regels dan. Let op: dit zorgt er dus wel voor dat gedeelten van deze Setup niet correct meer werken.
Nieuw in het thema is het bestand examples.php, deze bevat een aantal standaard codesnippets die gegenereerd zijn aan de hand van de hier ingevulde data.

Als je ideeen hebt om deze Setup uit te breiden dan kun je dat aangeven bij een PM.
